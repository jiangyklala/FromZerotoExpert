# FromZerotoExpert

## 注册:
### 用户名限制
+ 不能为空(前端)
+ 不能有空格
+ 不能有敏感词
+ 用户名唯一
### 密码限制
+ 不能为空(前端)
+ 长度
+ 强度(只能用三种字符中至少两种)
### 加密密码
+ 随机盐 + 真盐 + 双重MD5
## 登录
### 用户名限制
+ 不能为空(前端)
+ 是否不存在
    + 不符合"注册"时的限制
    + 未注册
### 密码限制
+ 不能为空(前端)
+ 错误
    + 不符合"注册"时的限制
    + 密码错误
### 自动登录
    登录时生成Cookie: fzteUser-sessionId, redis中存储: sessionId-userName,访问首页时根据是否有Cookie, 且Cookie中的sessionId在redis中是否过期实现自动登录
